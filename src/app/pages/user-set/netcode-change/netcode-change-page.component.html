網路連線代號設定規則

1、6-16位英數字(不可設定與身分證字號及登入密碼相同)。
2、不可使用相同的英、數字，如：111111、AAAAAA。
3、不可使用連續的英、數字，如：123456、ABCDEF。
4、使用英文字時，不區分大小寫。

請務必牢記新連線代號，日後若忘記此代號，需回本行營業單位辦理網路連線代號重設，或至網路銀行使用本行晶片金融卡線上重設密碼。
